Acerca de este proyecto

Este proyecto demuestra cómo implementar un sistema de autenticación basado en JWT (JSON Web Tokens) con diferentes roles de usuario:

  • Usuario: acceso básico
  • Moderador: acceso intermedio
  • Administrador: acceso completo
